# GameCard

Game Card, a component that represents one of 52 cards in a playing deck

#### Example

<Canvas of={GameCard.AceOfSpades} />

## Usage

```ts
import {GameCard, CardValue, CardSuite} from "tia-ui";

const Example = () => {
  return (
      <GameCard
        value={CardValue.Ace}
        suite=(CardSuite.Spades)
        isFlipped=(false)
        isFlippbable=(true)
      />
  );
};

export default Example;
```

#### Arguments

- **value** - `CardValue` enum value
- **suite** - `CardSuite` enum value
- **isFlipped** - `bool`, if card is flipped or not
- **isFlippbable** - `bool`if card can be flipped or not
